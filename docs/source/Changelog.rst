Changelog
=========

`OpenMMSLICER 1.0.0 <https://github.com/openmmslicer/openmmslicer/releases/tag/1.0.0>`_
-------------------------------------------------------------------------------------

* First stable version of OpenMMSLICER

=========

`OpenMMSLICER 3.0.0 <https://github.com/openmmslicer/openmmslicer/releases/tag/3.1.0>`_
-------------------------------------------------------------------------------------

* Removed ``integrators``
* Added ``alchemy\`` containing various system set-up tools (``align``, ``relative``, ``simulation``, ``softcore``)
* Added ``samplers\`` containing ``fast``, ``smc``, and ``st`` samplers, as well as ``misc`` for alchemical function and walker set-up
* Restructured ``smc``
* Added ``correlation_metrics``, ``fe_estimators``, ``interpolate``, ``minimise``, ``misc`` (logger), ``protocol``, ``reporters``, and ``transition_metrics``
* Restructured and expanded ``moves``
