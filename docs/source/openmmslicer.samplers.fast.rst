openmmslicer.samplers.fast module
=====================================

Implements the fully adaptive simulated tempering (FAST) sampler, which combines adaptive alchemical sequential Monte Carlo (AASMC) and irreversible simulated tempering (IST) with adaptive protocol optimisation.
